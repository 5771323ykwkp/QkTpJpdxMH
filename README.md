# Vue3纯前端后台管理系统 java10

## 项目概述

本项目是一款基于Vue3的纯前端后台管理系统，采用模块化设计，功能完善，页面美观，代码规范。系统无数据库，主要依赖前端技术实现数据管理，适用于学习Vue3、Pinia及权限管理开发。

## 技术栈

- 前端框架：Vue3
- UI组件库：Element Plus
- 构建工具：Vite
- 状态管理：Pinia
- 路由管理：Vue Router
- HTTP请求库：Axios
- 数据可视化：ECharts
- CSS预处理器：Sass

## 功能模块

系统包含以下10个页面：

- 登录页
- 注册页
- 仪表盘
- 用户管理
- 角色管理
- 权限管理
- 系统设置
- 403错误页
- 404错误页
- 各种弹窗组件

### 核心功能

- RBAC权限系统（用户-角色-权限）
- 数据图表（ECharts）
- 头像上传
- 父子组件通信（Prop、Emit）
- 常用指令（v-if、v-for、v-on、v-model、v-bind、v-permission）
- 计算属性（computed）
- 侦听器（watch）
- 响应式设计

### 功能特点

- 分页功能
- Pinia实现数据管理
- 支持PC端和移动端适配

## 系统角色

系统角色分为：用户、角色、权限。通过严格的权限控制，实现不同角色对系统资源的访问和管理。

## 运行环境

- 浏览器支持：现代浏览器（Chrome、Firefox、Safari等）
- 环境依赖：Node.js（版本建议12+）

## 目录结构

```
project/
├── public/                 # 静态资源目录
├── src/
│   ├── api/                # 接口请求目录
│   ├── assets/             # 资源目录（如图片、样式等）
│   ├── components/         # 组件目录
│   ├── views/              # 页面目录
│   ├── router/             # 路由配置目录
│   ├── store/              # 状态管理目录（Pinia）
│   ├── utils/              # 工具方法目录
│   ├── App.vue             # 主组件
│   └── main.js             # 入口文件
├── .gitignore              # Git忽略文件配置
├── package.json            # 项目依赖和配置
└── vite.config.js          # Vite配置文件
```

## 核心亮点

- 前端无数据库，纯前端实现数据管理
- 完整的权限控制系统，功能模块丰富
- 支持响应式布局，适配PC和移动端
- 代码规范，适合学习Vue3相关技术

## 部署步骤

1. 克隆项目代码到本地
2. 安装依赖（`npm install`）
3. 启动开发服务器（`npm run dev`）
4. 构建生产环境代码（`npm run build`）
5. 将构建后的代码部署至服务器

注意：本项目不含数据库，实际部署时可根据需求对接后端API。
